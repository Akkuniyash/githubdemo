command for including git repository:
  git clone https://github.com/Akkuniyash/githubdemo.git
change directory:
  cd filename
for checking status
  git status
Github folder->Copy Folder->working file
to change working file to staging area:
  git add filename   
to change from staging area to copy folder(local)   
  git commit -m "commit message"
to save the changes to github folder
  git push origin(to push in main)
